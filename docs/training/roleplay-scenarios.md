# Escenarios de role-play

El seed incluye siete categorías y catorce escenarios: dos para Ventas, Liderazgo, Negociación, Atención al cliente, Comunicación profesional, Emprendimiento y Finanzas/decisiones empresariales.

Cada escenario separa configuración pública y contexto privado. Una versión publicada es inmutable; los cambios requieren una versión nueva. Para añadir una categoría, crea `training_roleplay_categories`; para añadir un escenario, usa el panel editorial o crea escenario, versión y rúbrica en una transacción editorial.
