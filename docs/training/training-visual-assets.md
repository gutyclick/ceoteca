# Assets visuales

Se guarda `storage_path`, nunca una URL firmada. MIME permitido: PNG, JPEG, WebP y AVIF; dimensiones deben ser positivas y el texto alternativo tener al menos ocho caracteres. Orígenes y estado de copyright son catálogos cerrados.

Solo `approved` se entrega a usuarios. `restricted`, `rejected` y `needs_review` permanecen fuera de consultas normales. No existe upload UI en este bloque.
