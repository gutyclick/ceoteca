# Taxonomía de CEOTECA Training

La jerarquía oficial es `categoría → subcategoría → habilidad → concepto → nivel cognitivo → formato → ejercicio`.

- La categoría responde qué área de negocio se entrena.
- La habilidad describe una conducta observable y medible.
- El concepto contiene el conocimiento mínimo necesario.
- El nivel cognitivo indica la profundidad del aprendizaje.
- El formato define cómo se practica.
- El ejercicio es la unidad que ejecuta el motor de sesiones existente.

Las entidades se archivan; no se eliminan cuando tienen dependencias. Los slugs son estables, únicos y conservan alias al cambiar.
