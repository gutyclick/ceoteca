# Panel editorial de role-play

`/admin/training/roleplay` usa los roles editoriales existentes. Editores crean borradores; reviewers revisan; administradores/publicadores publican. La API valida personaje, objetivo, apertura, contexto privado, safety y rúbrica antes de publicar.

Las versiones publicadas son inmutables. Preview real deberá pedir confirmación, nunca consumir cuota ni actualizar dominio y registrar uso editorial.
