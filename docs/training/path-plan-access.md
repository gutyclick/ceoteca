# Acceso de rutas por plan

El plan efectivo se obtiene de la suscripción en servidor y se normaliza a Free, Pro o Unlimited. El cliente no envía plan, cuotas ni permisos. Free puede previsualizar rutas bloqueadas, pero no iniciar contenido Pro; Pro accede a Free y Pro; Unlimited accede a todos los niveles publicados.

El bloqueo se valida tanto al iniciar una ruta como al iniciar un item. La interfaz explica el requisito y puede ofrecer contenido alternativo gratuito, pero nunca sustituye la validación del servidor.
