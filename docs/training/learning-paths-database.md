# Rutas de aprendizaje en base de datos

Las ocho rutas iniciales son seeds publicados. Cada ruta contiene módulos ordenados por `sort_order` y slug único por ruta. Los items usan una unión discriminada: su `item_type` exige exactamente una FK compatible.

Free puede explorar rutas; Pro accede al entrenamiento estándar; Unlimited incluye contenido avanzado. El acceso final siempre se deriva en servidor. Para añadir una ruta, crear ruta y módulos en una migración idempotente, asociar habilidades/categorías y validar módulos vacíos e items.
