# Analítica de role-play

Medir catálogo, inicio, primer mensaje, consumo, pausa, reanudación, finalización, evaluación, abandono, seguridad y cuota agotada. Propiedades permitidas: categoría, escenario, dificultad, plan, turnos, duración, resultado, score bucket, cuota, modelo y costo bucket.

No enviar el texto del transcript a analítica externa. La duración no se interpreta por sí sola como aprendizaje.
