# Formatos de entrenamiento

Los ocho formatos son análisis visual, caso, respuesta escrita, role-play, constructor guiado, diagnóstico, simulación de decisiones y práctica determinista. `training_exercise_formats` permite varios formatos, con un índice parcial que garantiza como máximo uno principal. Este bloque no implementa renderers.

Los formatos oficiales son análisis visual, análisis de casos, respuesta escrita, role-play conversacional, constructor guiado, diagnóstico, simulación de decisiones y práctica determinista.

Un formato describe cómo se entrena y nunca debe usarse como categoría. `training_exercise_formats` permite varios formatos por ejercicio, pero garantiza un único formato principal.
