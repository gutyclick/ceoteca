# Ejercicios de mensajes

Los tipos son respuesta a mensaje, comparación, ajuste de tono, respuesta a objeción, reescritura de correo y diagnóstico conversacional.

El contenido usa situaciones y nombres ficticios. Las respuestas abiertas pasan por la evaluación existente; las comparaciones deterministas usan reglas editoriales. El feedback explica criterio, tono y siguiente paso sin presentar una única redacción como verdad universal.
