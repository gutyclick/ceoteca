# Análisis visual

Los ejercicios visuales soportan selección, comparación, diagnóstico, anotación y ordenamiento. Todo asset exige ruta de Storage, dimensiones, MIME permitido, texto alternativo, origen y estado de derechos.

Solo los assets con `copyright_status = approved` son legibles por usuarios. Cada ejercicio debe ofrecer una alternativa textual suficiente. No se publican marcas, portadas oficiales ni material sin licencia.
