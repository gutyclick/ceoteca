# Planes de role-play

Free puede explorar escenarios, ver objetivos y usar ejercicios deterministas, pero la API rechaza inicios. Pro/Fundador dispone de dos inicios por mes calendario y de Fundamentos/Aplicación. Unlimited accede a Fundamentos, Aplicación, Avanzado y Experto cuando las flags lo permiten.

El cliente nunca envía el plan ni el saldo. `getEffectiveSubscriptionForUser` resuelve el plan efectivo en servidor.
