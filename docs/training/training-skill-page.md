# Página de habilidad

`/ejercicios/habilidades/[skillSlug]` muestra categoría, subcategoría, dificultad, dominio general, dominio cognitivo, conceptos, formatos publicados y repaso pendiente.

`StartSkillTrainingButton` solicita una recomendación adaptativa centrada en la habilidad, la acepta y navega a la sesión creada por el motor existente. La API determina identidad, plan y elegibilidad. Un slug inválido o no publicado devuelve 404.
