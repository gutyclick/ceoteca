# Modos de Training

Los modos son filtros de descubrimiento, no una taxonomía alternativa:

- `analiza`: análisis visual, casos, diagnóstico y simulación de decisiones.
- `construye`: respuesta escrita y constructores guiados.
- `practica`: escenarios y práctica determinista; role-play solo cuando su flag lo permite.

La selección se conserva con `?mode=analiza|construye|practica`. Los conteos se calculan a partir de ejercicios publicados.
