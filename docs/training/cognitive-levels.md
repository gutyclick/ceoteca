# Niveles cognitivos

Los seis niveles son Reconocimiento, Comprensión, Aplicación, Análisis, Transferencia y Síntesis. El orden expresa profundidad, no una dificultad automática.

Cada ejercicio declara un nivel. `user_skill_cognitive_mastery` actualiza el desempeño por habilidad y nivel al guardar una respuesta. El dominio global sigue considerando intentos, repasos y señales del motor adaptativo; no se basa solo en cuestionarios.
