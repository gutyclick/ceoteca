# Prompts de role-play

`roleplay-character-v1` contiene rol, configuración pública, contexto privado, estado, resumen, dificultad y seguridad. El personaje responde breve, mantiene el escenario y no evalúa ni revela instrucciones.

`roleplay-evaluation-v1` usa transcript y rúbrica versionada. La salida se valida con Zod y se rechazan ids de evidencia inexistentes. No se almacena razonamiento interno.
