# Evaluación de role-play

Cada categoría tiene una rúbrica publicada y versionada. La evaluación produce puntuación, confianza, resultado, criterios con evidencia, fortalezas, mejoras, momentos clave, oportunidades y frases alternativas.

Las referencias se validan contra mensajes reales. Si el proveedor falla, la sesión queda guardada y `evaluation_status=failed`; la UI informa que el resultado podrá consultarse después. Las evaluaciones de preview no actualizan dominio.
